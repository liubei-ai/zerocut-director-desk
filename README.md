# ZeroCut Director's Desk

基于浏览器的 3D 分镜/场景编排工具，用于影视、漫画、动画预制作阶段的场景布局与姿势设计。无需安装，打开即用。

[![Open in Bolt](https://bolt.new/static/open-in-bolt.svg)](https://bolt.new/~/sb1-zpdrxqzn)

---

## 功能概览

| 功能 | 说明 |
|---|---|
| 3D 场景视口 | Three.js 渲染，支持轨道旋转、平移、缩放 |
| 人形角色 | 放置多个角色，独立设置颜色与位置 |
| 8 种预设姿势 | 站立、T形、行走、坐姿、举臂、动作、蹲伏、指向 |
| 自由画线 | 在画面上叠加手绘标注，可调颜色与粗细 |
| 背景图 | 上传参考图铺满场景背景 |
| 导出 PNG | 全图导出或框选区域导出，支持透明背景 |

## 快速上手

### 视角操控

| 操作 | 效果 |
|---|---|
| 左键拖拽 | 旋转视角 |
| 右键拖拽 | 平移视角 |
| 滚轮 | 缩放（也可拖动左下角滑条） |

### 添加与编辑角色

1. 点击工具栏 **角色** 按钮，光标变为十字准星
2. 在场景地面任意位置点击，角色即被放置并自动选中
3. 右侧属性面板中可切换 **姿势**、更改 **颜色**、调整 **位置 / 旋转**
4. **`Ctrl` + 拖拽角色** 可在 3D 场景中自由移动该角色
5. 左侧角色列表支持重命名与删除

**可选姿势（8 种）**

`站立` · `T形站立` · `行走` · `坐姿` · `举臂` · `动作姿势` · `蹲伏` · `指向`

### 画线标注

1. 点击工具栏 **画线** 按钮（视角旋转暂时锁定）
2. 在画布上拖拽鼠标，沿轨迹生成曲线
3. 工具栏中调整 **颜色** 和 **粗细**（1–20）
4. 点击 **清除** 删除全部线条
5. 点击 **选择** 退出画线模式，恢复视角操控

### 背景图

- 点击工具栏 **背景图** 按钮，上传本地 JPG / PNG / WebP
- 背景加载后地面网格自动隐藏，角色渲染于背景之上
- 点击缩略图旁的 **×** 移除背景，恢复默认场景

### 导出 PNG

1. 点击工具栏 **导出 PNG** 进入截图模式
2. **导出全图**：右上角按钮，一键导出完整画面
3. **框选导出**：在画布上拖拽选定区域，松开后点击「导出选区 PNG」
4. 无背景图时导出透明背景；背景图与手绘线条均会一并包含

## 本地开发

```bash
npm install
npm run dev
```

构建生产版本：

```bash
npm run build
```

---

## 技术栈

- **React 18** + **TypeScript**
- **Three.js** — 3D 渲染
- **Zustand** — 全局状态管理
- **Tailwind CSS** — 样式
- **Vite** — 构建工具
